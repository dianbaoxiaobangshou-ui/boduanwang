**项目简介**：

本项目使用 **欧意公开API** 获取加密货币合约的行情数据，支持多币种监控，并能根据用户自定义的周期（如12小时K线、日线等）进行自动分析和信号推送。根据历史K线数据的走势，脚本会判断市场趋势，并通过 **纸飞机机器人** 发送通知，帮助用户实时掌握市场动态。

### 核心功能：

1. **加密货币行情获取**：

   * 支持监控多个币种对（如 BTCUSDT、ETHUSDT 等）及其合约类型。
   * 用户可以自定义K线周期（如 12小时、日线等）。

2. **市场趋势判断**：

   * 脚本基于历史K线数据进行市场走势判断走势强弱，适合做波段。
   * 判断规则：

     * **阳线**：当前K线的收盘价高于开盘价。
     * **阴线**：当前K线的收盘价低于开盘价。
   * 判断逻辑：

     * **阴阴阳排列**：市场趋强，提示“走强”。
     * **阳阳阴排列**：市场趋弱，提示“走弱”。
![图片描述](https://github.com/dianbaoxiaobangshou-ui/-/raw/main/1.jpg)

    

3. **时间触发机制**：

   * 仅在当前K线周期的收盘时刻进行判断，避免不必要的频繁计算。

4. **消息推送**：

   * 在满足条件时，脚本会自动推送消息至用户指定的 **纸飞机机器人**。
   * 消息内容包括：

     * 当前币种及其合约类型
     * 当前K线周期
     * 触发的信号（如“走强”或“走弱”）
     * 当前K线的具体数值与时间戳

5. **错误处理与重试机制**：

   * 如果API请求失败或返回数据格式错误，脚本会记录错误日志并继续执行。
   * 如果消息发送失败，脚本会自动进行重试，确保推送稳定。

### 配置要求：

* 用户需提供想要监控的币种对（如BTCUSDT、ETHUSDT等）及其合约类型。
* 支持自定义K线周期（如12小时、日线等）。
* 需提供 **纸飞机机器人的API密钥** 和 **chat_id** 以便推送通知。
